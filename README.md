#  Alimentation Durable & Nutrition Intelligente

## Overview
Ce projet est développé dans le cadre du module Projet Technologies Web (2A) à Esprit School of Engineering (année universitaire 2025–2026).

L’objectif est de proposer une application web qui aide les utilisateurs à adopter une alimentation durable en réduisant le gaspillage alimentaire et en favorisant des recettes intelligentes.

---

##  Features
- Suggestion de recettes à partir des restes
- Gestion des ingrédients
- Sensibilisation écologique
- Statistiques d’utilisation

## Getting Started

### 1. Cloner le projet
```bash
git clone https://github.com/
```

### 2. Entrer dans le dossier
```bash
cd Alimentation durable 
```

### 3. Installer les dépendances
```bash
npm install
```

### 4. Lancer le projet
```bash
npm start
```
